## 1. 수치형  

파이썬은 모든것이 객체이다. 
10은 수치값이 아닌 객체이다 => 주소값이 변수에 세팅된거라서 


```py

# 정수
a = 10
print( a )

# 소수 
a = 1.1
print( a )




# 일반적인 연산자는 대동소이하다.

# 연산자도 우선순위가 있다
# *, / 가 +, - 우선한다 => 연산자 우선순위를 다 알면 좋으나,
# 괄호를 이용하여 () 우선순위 연산을 그룹으로 묶어준다
(1+2+3+4) * 2 , 1+2+3+4 * 2
```
