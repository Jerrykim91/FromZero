
# 변수의 사용 범위 

## 전역변수(global variable) 
- 프로그램 전체에서 동작 

<br>

###  전역변수 예제 
```py
# 전역변수 예제 

x = 100  
def val():
    print(x) # 전역변수 

val()
print(x)

# Output
# >>> 100
# >>> 100

```    


## 지역변수(??? variable) 
- 함수 내부에서 동작

<br>

###  지역변수 예제 


```py
# 지역변수 예제 

x = 100  
def val():
    x = 10  # 지역변수 
    print(x) 

val()
print(x)

# Output
# >>> 10
# >>> 100

```

### 함수안에서 전역 변수의 값을 변경

함수안에서 전역 변수의 값을 변경하려면 `global` 을 사용 

```py
# 지역변수 예제 

x = 100  
def val():
    global x # 전역 변수로 선언 
    x = 10   # 지역변수x -> 전역변수로 변경  
    print(x) 

val()
print(x)

# Output
# >>> 100
# >>> 100

```
