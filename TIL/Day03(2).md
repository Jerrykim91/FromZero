### 벡터 


`:`와  `c()`를 이용하여 주어진 인수로 벡터를 생성가능하다. 
`seq()`을 이용하면 순차적으로 벡터를 이용가능하다. 
`rep()` 은 반복적이 벡터를 생성 가능하다. 

```r
# 0~10까지 2씩 증가하는 코드 
seq(from=0, to=10, by=2) 
seq(0, 10, 2) 

# 벡터 단위 반복
rep(c(1:10), times = 2)

# 개별 반복 
rep(c(1:10), each = 2)
```
